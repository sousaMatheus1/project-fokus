# Fokus Project - Pomodoro Timer

This project is a time management application based on the Pomodoro technique, developed to practice DOM manipulation and data persistence in the browser.

## 📚 Course Details (Alura Training)

### 1. JavaScript: Manipulating elements in the DOM
* **Workload:** 6 hours
* **Challenge:** Building the core of the Fokus project to switch contexts (focus/rest) by manipulating phrases, images, and background colors.
* **Solution:** Implemented JavaScript logic to toggle CSS classes, trigger audio files, and manage timer functionalities.
* **Result:** A functional interface with an interactive timer and real-time visual feedback.

### 2. JavaScript: Exploring element manipulation and localStorage
* **Workload:** 8 hours
* **Challenge:** Advancing the application by adding a task management system (CRUD - Create, Read, Update, Delete).
* **Solution:** Utilized the `localStorage` API for data persistence and created customized events to handle task creation, editing, and deletion. Applied debugging techniques to refine the code.
* **Result:** A complete application with a persistent task list, allowing for efficient workflow management.
